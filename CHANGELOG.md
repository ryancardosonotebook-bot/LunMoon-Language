# CHANGELOG

## [1.0.0] - 2026-03-24
### Added
- **Lexer**: a component to tokenize input source code.
- **Parser**: processes tokens and builds an abstract syntax tree.
- **Interpreter**: executes the code based on the syntax tree.
- **Compiler**: compiles code to target language.
- **Terminal Commands**: implemented `cd`, `pwd`, and `ls` commands for terminal interaction.
- **Examples**: provided sample source code for testing features.
- **Documentation**: comprehensive documentation for usage, including installation and examples.
